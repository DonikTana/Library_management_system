# 📚 Smart Library Management System

## 🔹 Description

This project is a **Smart Library Management System** developed as a minor project. It aims to digitalize library operations such as book management, borrowing, and study hall seat reservations.

The system also includes an **analytics dashboard** to visualize library usage and improve decision-making.

---

## 🚀 Features

### 👨‍💼 Admin
- Add, update, and manage books  
- View all borrowed books  
- Manage study hall seat reservations  
- Access analytics dashboard  

### 🎓 Student
- Register and login securely  
- Search available books  
- Borrow and return books  
- Reserve and release study hall seats  
- View personal activity  

---

## 🪑 Study Hall Seat Reservation
- Interactive seat layout (grid-based UI)  
- Color-coded seats:
  - 🟢 Available  
  - 🔴 Reserved by others  
  - 🔵 Reserved by current user  
- One seat per student restriction  
- Admin override for seat release  

---

## 📊 Analytics Dashboard
- Most borrowed books (Bar chart)  
- Weekly student activity (Line chart)  
- Library usage insights
